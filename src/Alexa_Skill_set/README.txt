Die beigelegten Screenshots sind Darstellungen aus dem Amazon Developer-Portal.
Dieses Portal wird von Amazon zur Verfügung gestellt um eigene Skills zu entwickeln. 
Der beigelegte Code kann in dem Portal in jedem Skill hinzugefügt werden. 
Das Portal bündelt die verbalen Anfragen in Absichten (Intents) welche als JSON-Objekte übergeben werden.
Die Objekte können entweder an AWS (Amazon Web Services) Lambda oder über HTTPS an andere Plattformen (wie zB Bluemix) übergeben werden.
Dort findet die eigentliche Verarbeitung statt und die Antworten werden generiert.
